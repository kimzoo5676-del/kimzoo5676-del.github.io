01 / About

근거 위에서 만들고,
쉽게 전합니다

복잡한 정보를 사용자가 이해할 수 있는 형태로 바꾸는 일을 선호합니다. 실제 데이터를 확인하고, AI를 실무에 맞게 적용하며, 설계부터 배포까지 서비스 전 과정을 다룹니다.

/ 01

데이터로 검증하는 문제 해결

가정을 코드로 옮기기 전에 실제 데이터를 확인합니다. 추측으로 만든 로직은 실제 데이터 앞에서 어긋나기 쉽다는 것을 경험했기에, 문제가 생기면 감이 아니라 데이터를 먼저 열어봅니다.

/ 02

실무에 맞게 적용하는 AI

AI를 목적에 맞는 형태로 설계합니다. 반복되는 가공 작업은 배치로 미리 처리해 응답 속도와 비용을 낮추고, 실시간 응답은 답변 근거를 명확한 데이터로 한정해 신뢰할 수 있는 결과를 만듭니다.

/ 03

설계부터 배포까지

DB 모델링과 API 설계부터 CI/CD 구축, 클라우드 배포와 도메인 연결까지, 서비스가 만들어지고 운영되는 생명주기 전체를 경험하고 전체 흐름을 이해한 상태에서 개발합니다.

02

가정하지 않는다

"아마 이럴 것이다"로 넘어가지 않고, 실제 데이터와 동작을 직접 확인한 뒤 다음 단계로 갑니다.

03

반복은 자동화한다

빌드·배포처럼 반복되는 일은 CI/CD와 배치로 자동화해, 실수를 줄이고 만드는 데 집중합니다.

03 / AI Workflow

AI를 다루는 방식

모두가 AI를 쓰는 시대에는, 어떻게 쓰는지가 결과의 차이를 만든다고 생각합니다. AI를 단순한 코드 생성기가 아니라, 명확한 규칙 위에서 움직이는 협업 파트너로 사용합니다.

01

합의

02

규칙화

03

실행
