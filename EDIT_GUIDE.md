# 포트폴리오 편집 가이드

이 폴더는 김주우님의 포트폴리오 웹사이트입니다. 디자인은 그대로 두고,
**내용만 채워 넣으면** 완성됩니다.

## 1. 채우는 법 (핵심)

모든 페이지에서 채울 자리는 **【 】** 기호로 표시되어 있습니다.

- 텍스트 편집기(메모장, VS Code 등)로 `.html` 파일을 엽니다.
- `【` 를 검색(Ctrl+F / Cmd+F)하면 채울 곳을 하나씩 찾을 수 있습니다.
- **【 】 기호까지 포함해서** 안의 안내문을 지우고 실제 내용을 적습니다.
  - 예) `<span class="hero-label">【직무 / 포지션】</span>`
    → `<span class="hero-label">프론트엔드 개발자</span>`

미리 보려면 `index.html` 파일을 더블클릭해 브라우저로 열면 됩니다.

## 3. 자주 묻는 편집

**숙련도 막대 (skills.html)**
숫자와 막대 길이 두 군데를 **같은 값**으로 맞춰야 합니다.

```html
<span class="skill-item-level">80%</span>      ← 표시되는 숫자
<div class="skill-bar-fill" style="width: 80%"></div>   ← 막대 길이
```

**항목 개수 늘리기/줄이기**
프로젝트 카드는 `<article class="project-card">` ... `</article>` 한 덩어리를,
스킬·타임라인 항목도 비슷하게 한 덩어리를 통째로 복사하거나 삭제하면 됩니다.

**링크 연결**
GitHub/LinkedIn/블로그 버튼은 `href="#"` 의 `#` 자리에 실제 주소를 넣으세요.
예) `<a href="https://github.com/내아이디">`
