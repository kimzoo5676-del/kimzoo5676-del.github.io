SSAFY RACE

SSAFY RACE — 자율주행 시뮬레이션
기간
2026.05 — 2026.06
역할
팀 프로젝트 · 주행 알고리즘
한 줄 요약
실시간 차량·도로 센싱 정보로 조향과 가감속을 제어해, 트랙을 가장 빠르고 안정적으로 완주하는 자율주행 알고리즘
성과
전체 278팀 중 2위
Environment
Python Microsoft AirSim Driving API
Algorithm
실시간 조향·속도 제어 빈 공간 탐색 회피 충돌 복구 상태 머신
AI 협업
가설 검정 루프 단일 변수 실험 인계 문서

배경 & 문제

삼성청년SW·AI아카데미에서 열린 자율주행 시뮬레이션 알고리즘 경진대회입니다(777명 · 278팀 참가). 실시간으로 주어지는 차량·도로 정보를 기반으로 조향·가속·감속을 제어해, 장애물이 배치된 다양한 트랙을 완주해야 합니다. 막연히 "빠르게"가 아니라 완주를 보장하면서 랩타임을 단축한다로 목표를 정의하고, 1위 기록과의 격차를 수치로 추적하며 접근했습니다.

한 일

  • 전방 커브 각도와 현재 주행 각도의 차이로 조향을 계산하고, 속도가 빠를수록 더 먼 지점을 참조하는 실시간 주행 제어를 구현했습니다. 급커브에서는 곡률 반경 기반 계산으로 전환하고 감속을 함께 걸어 이탈을 막았습니다.
  • 장애물 배치마다 회피 코드를 분기하던 초기 설계를 버리고, 도로를 0.1m 간격의 후보선으로 보고 막힌 구간을 지운 뒤 가장 가까운 빈 공간을 향하는 단일 방식으로 재설계했습니다. 느린 상대 차량도 동적 장애물로 편입해, 예상 못 한 배치까지 별도 분기 없이 처리합니다.
  • 충돌·역주행 시 자동으로 후진 탈출 후 재출발하는 복구 상태 머신을 구현해, 어떤 상황에서도 완주가 끊기지 않게 했습니다.
  • AI와는 가설 → 구현 → 측정 → 판단 루프로 협업했습니다. 한 번에 하나의 변수만 바꾸는 단일 변수 실험 원칙을 지키고, AI는 가설과 코드를 제안하되 검토·반영·시뮬레이션 실행·채택 판단은 직접 수행했으며, 세션이 바뀌어도 맥락이 이어지도록 베이스 코드·검증된 인과·기각된 가설·다음 결정지점을 담은 인계 문서를 관리했습니다.

결과 & 배운 점

전체 278팀 중 2위를 달성했습니다. 같은 코드도 실행 상황에 따라 기록이 초 단위로 흔들리는 것을 확인하고, "작은 차이는 노이즈, 뚜렷한 변화만 유의미"라는 측정 기준을 세워 잘못된 미세 튜닝을 줄였습니다. 회피 여유 확대, 회피 구간 감속처럼 기각된 가설도 지우지 않고 기록으로 남겨 같은 시행착오를 반복하지 않았습니다. 감이 아닌 데이터 기준으로 판단하는 가설 검정 루프를 실전에서 검증한 프로젝트입니다.
