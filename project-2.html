SSAFY RACE

기간
2026.05 — 2026.06
역할
팀 프로젝트 · 주행 알고리즘
한 줄 요약
실시간 차량·도로 센싱 정보로 조향과 가감속을 제어해, 트랙을 가장 빠르고 안정적으로 완주하는 자율주행 알고리즘
성과
전체 278팀 중 2위
Environment
Python Microsoft AirSim Driving API
Algorithm
실시간 조향·속도 제어 빈 공간 탐색 회피 충돌 복구 상태 머신
AI 협업
가설 검정 루프 단일 변수 실험 인계 문서

배경 & 문제

삼성청년SW·AI아카데미에서 열린 자율주행 시뮬레이션 알고리즘 경진대회입니다(777명 · 278팀 참가). 실시간으로 주어지는 차량·도로 정보를 기반으로 조향·가속·감속을 제어해, 장애물이 배치된 다양한 트랙을 완주해야 합니다. 막연히 "빠르게"가 아니라 완주를 보장하면서 랩타임을 단축한다로 목표를 정의하고, 1위 기록과의 격차를 수치로 추적하며 접근했습니다.

한 일 — AI 협업 워크플로우

  • 합의 — 기획·목표·제약을 AI와 먼저 합의했습니다. "완주를 보장하면서 랩타임을 단축한다"로 목표를 명시하고 1위와의 기록 격차를 수치로 제시했으며, 대회 규칙·API 명세·트랙 스펙·제어 변수 범위를 전달해 의도와 다른 방향으로 접근하지 않게 했습니다. AI의 분석은 참고하되, 결정권은 제가 가졌습니다.
  • 규칙화 — 합의된 내용을 AI가 지켜야 할 규칙 문서로 고정했습니다. AI는 코드를 제시만 하고 반영은 제가 검토 후 직접 수행, 한 번에 하나의 가설만 바꾸는 단일 변수 실험으로 인과를 구분, 세션이 바뀌어도 맥락이 이어지도록 베이스 코드·검증된 인과·기각된 가설·다음 결정지점을 담은 인계 문서를 관리했습니다.
  • 실행 — 가설 → 구현 → 측정 → 판단 루프를 반복했습니다. 장애물 배치마다 분기하던 회피 코드를 가장 가까운 빈 공간을 향하는 단일 방식으로 통일해 예상 못 한 배치까지 처리했고, 회피 여유 확대·회피 구간 감속 같은 가설은 측정 결과로 기각했습니다. 시뮬레이션 실행과 채택 판단은 제가 담당했습니다.
  • 회고 — 기각한 실험도 지우지 않고 단서로 남겨 다음 실험이 같은 길을 반복하지 않게 했고, "작은 차이는 노이즈, 뚜렷한 변화만 유의미"라는 측정 기준을 세워 판단을 감이 아닌 데이터 기준으로 수행했습니다.

결과 & 배운 점

전체 278팀 중 2위를 달성했습니다. 가설을 설정하고, 기각 기준을 정한 뒤, 측정과 검정으로 채택·기각을 판단하고, 그 이력을 문서로 누적하는 가설 검정 사이클을 반복해 얻은 결과입니다. "작은 차이는 노이즈, 유의미한 변화만 반영한다"는 판단 기준으로 노이즈에 맞추는 과적합성 튜닝을 배제했고, 데이터 기반 가설 검정이 실제 성과로 이어짐을 확인했습니다.
